Instrucciones:

En esta actividad continuarás haciendo mejoras a tu proyecto, por ello ahora que has aprendido sobre el modelo de cajas, es tu turno de implementarlo.

 

En el módulo 1, definiste tu proyecto y dentro creaste un archivo index.html en visual studio code, ahora trabajarás sobre este mismo documento.

 

Dentro del documento index.html:

 

  • Añade la etiqueta <style> </style> dentro de las etiquetas de <body> </body> Por ejemplo:

 

<!DOCTYPE html>

<html>

  <head>

     <title> Mi Proyecto </title>

  </head>

  <body>

    <style>

      /* Dentro de esta etiqueta se definen los estilos a los ids y clases */

    </style>

    <img src=”mi_imagen.png”>

  </body>

</html>

 

  • Definir una clase para altura y anchura de imágenes. Por ejemplo:

 

<!DOCTYPE html>

<html>

  <head>

     <title> Mi Proyecto </title>

  </head>

  <body>

    <style>

      /* Dentro de esta etiqueta se definen los estilos a los ids y clases */

      .my-image {

        width: 200px;

        height: 200px;

      }

    </style>

    <img src=”mi_imagen.png” class=”my-image”>

  </body>

</html>

 

  • Generar una clase que se añada a una etiqueta <button> o <a> que tenga un color de fondo, un margen ovalado (margin-radius)
  • Añadir una clase (class) que haga que mi color de botón cambie de color cuando el mouse se posicione sobre el elemento (tip: revisa la documentación de hover https://developer.mozilla.org/en-US/docs/Web/CSS/:hover)
  • Cambia el color de fondo de tu página, navbar o encabezados
  • Generar una barra de navegación horizontal para ser usada como tu menú, te recomendamos que te inspires en https://www.w3schools.com/csS/css_navbar_horizontal.asp

 

 

Ahora es momento que hagamos una mejor práctica, para ello:

 

  • Dentro de tu proyecto crea un archivo llamada style.css
  • Copia y pega en style.css el contenido que hayas puesto dentro de las etiquetas <style></style> de tu archivo index.html. Y guarda tu documento. Ejemplo:

 

      .my-image {

        width: 200px;

        height: 200px;

      }

 

  • Elimina las etiquetas de <style> Ejemplo:

 

<!DOCTYPE html>

<html>

  <head>

     <title> Mi Proyecto </title>

  </head>

  <body>

    <img src=”mi_imagen.png” class=”my-image”>

  </body>

</html>

 

  • En cuanto se eliminan las etiquetas, volvemos a ver un index.html sin estilos, así que para ello hay que llamar a nuestro archivo style.css, para ello dentro de la etiqueta <head> agrega una etiqueta <link> Ejemplo (tip: https://www.w3schools.com/tags/tag_link.asp)

 

<!DOCTYPE html>

<html>

  <head>

     <title> Mi Proyecto </title>

     <link rel=”stylesheet” href=”style.css”>

  </head>

  <body>

    <img src=”mi_imagen.png” class=”my-image”>

  </body>

</html>

  • Con esto hemos separado nuestro código HTML y CSS, esta es una buena práctica que con el paso del tiempo permite que el código se pueda mantener actualizado.

 

Por último, recuerda que el objetivo de esta actividad es que tu proyecto se vea armonioso, siempre piensa que tus usuarios la van a utilizar y que los estilos ayudarán a hacer que su experiencia en el sitio sea más placentera, por lo que si necesitas más estilos los puedes añadir dentro de style.css y en index.html los utilizas a través de los atributos class o id.

Resguarda la información en tu repositorio, ya que esto forma parte de la evaluación del proyecto final.

¡Éxito!
